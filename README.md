# **App Name**: EduAI

## Core Features:

- Login Page: Develop a login page with options for 'forgot password', 'login', and 'register'.
- Text Query Processing: Accept text input and provide relevant answers using a generative AI tool.
- Voice Query Processing: Process voice queries and generate spoken responses using a generative AI tool.
- Image Query Processing: Analyze image inputs and use a generative AI tool to give contextually relevant information.
- Contextual Response Display: Display responses in a clear and engaging format, including text, charts, and diagrams.
- Engagement Monitoring: Track student engagement levels and provide insights to educators.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF), providing a calming and intellectual feel conducive to learning.
- Background color: Light cyan (#E0FFFF), offering a clean and non-distracting backdrop.
- Accent color: Royal blue (#4169E1), used for interactive elements and calls to action.
- Body text font: 'PT Sans' (sans-serif) for readability and a modern touch.
- Headline font: 'Playfair' (serif) for elegance.
- Use clean, simple icons to represent different functions and query types.
- Maintain a clean, intuitive layout to enhance usability and focus on educational content.
- Use subtle animations to provide feedback and guide users through the learning process.
